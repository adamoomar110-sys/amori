# Amori

Amori is a multi-platform application designed for personal management and interaction.

## Features
- **Web Mode**: Localized web interface for desktop use.
- **Mobile Mode**: Remote access via Ngrok with QR code generation for mobile devices.
- **AI Integration**: Backend services powered by advanced AI models.

## Getting Started

### Prerequisites
- Python 3.12+
- Node.js & npm

### Installation
1. Install backend dependencies:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
2. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```

### Running the App
Use the `start_app.py` script in the root directory:
```bash
python start_app.py --mode web
```
or use the desktop shortcuts provided.
